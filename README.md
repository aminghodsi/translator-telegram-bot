# 🌍 Translator Telegram Bot

A Telegram bot built with Python and Aiogram that can:

- 🌐 Translate text between multiple languages
- 🎤 Convert voice messages to text using Faster-Whisper
- ⚡️ Fast and lightweight
- 🤖 Easy to use

## Libraries

- Python
- Aiogram 3
- Faster-Whisper
- Deep Translator
- Pydub


## Run

python main.py
